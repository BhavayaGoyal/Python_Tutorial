# Object types / data types 

- Number : 1234, 3.12, 3+4j, 0b111, Decimal(), Fraction()

- String: 'spam', "Bob's", b'a\x01c', u'ap\xc4m'

- List: [1, [2, "Three"], 4.5], list(range(10))

- Tuple: (1, 'spam', 4, 'U'), tuple('abc), namedtuple('Point', 'x y')

-Dictionary: {'food': 'spam', 'taste': 'yum'}, dict(hours = 10)

- Set: set('abc'), {'a', 'b', 'c'}

- File: open('egg.txt'), open(r'C:\ham.bin', 'wb')

-Boolean: True, False

-None: None

-Functions, modules, classes

- Advance: Decorators, Generators, Iterators, Meta Programming
